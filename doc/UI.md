# 视觉与交互设计规格 (UI Specification)

> 全站基础配色与动效规范见 `doc/design-guidelines.md`（动森式配色氛围 + 苹果式过渡，作为默认规则，不在首页单独展示）。

## 1. 核心视觉理念：玻璃自然风
- **质感**：全站应用毛玻璃效果 (Backdrop Blur)，强化层级感。
- **圆角**：统一大圆角 (16px+)，营造现代、友好的氛围。
- **色彩**：
  - 基础色：奶白色背景 (#F9FAFB)，深灰色文字 (#1F2937)。
  - 毕业主题：天蓝 (#0EA5E9) + 柠檬黄 (#FDE047)。
  - 春节主题：现代朱红 (#DC2626) + 琥珀金 (#F59E0B)。

## 2. 页面布局
### 编辑器
- **画布**：居中显示，阴影深邃，模拟真实纸张/相册浮动。
- **左栏**：侧边悬浮式玻璃条，图标加文字标签。
- **右栏**：按需出现的浮动抽屉，不遮挡画布中心。

## 3. 交互细节
- **反馈**：所有交互（拖拽、悬浮、点击）必须有平滑的状态过渡 (motion-apple)。
- **引导**：Layout 圈选时，选中的格子有发光描边效果。
- **导出**：点击导出按钮时，展示一个精致的进度条动画。

## 4. 字体
- 标题：现代粗体 (e.g., "HarmonyOS Sans", "Inter")。
- 装饰文字：支持 1-2 种手写体 (e.g., "Kaiti" 或内置的 Google Fonts 手写体)。


# 页面设计
我们一共三个界面

1. 首页/about - 保持目前这个首页

2. 开始创造
   - 这个页面就是用户开始创作的页面，用户可以在这个页面上选择合适的模板，然后在模板上，添加合适的图片即可完成创作。

进入之后会分成左右两个部分，左侧是工具栏，右侧是预览和展示栏目。

左侧标签卡（功能栏）
  选择对应的标签（顶上一排标签卡）。
  标签卡：下方展示可选的内容就分成两个列。

  1. 相册选择（类似选择一个主题风格的相册）+尺寸选择（A4/A5等）
    当进入界面的时候，默认预览界面默认显示相册主题的选择。
    目前有以下的主题
    1. 青春纪念（校园、青春、学生、懵懂、纯真）
    2. 春节团圆（喜庆、热闹、红火、团圆）
    3. 动森梦幻
    4. 默认（什么都没有）

  2. 照片墙选择（相册里面每一个界面选择 样式/主题）
      这里提供默认的布局可以选择
        比如爱心布局：可能是 30*30，然后呈现爱心各自
        比如九宫格：9*9 ，然后呈现9个格子，每个格子都有一个图片

  3. 装饰选择（可以给每一个照片墙添加一些装饰，比如文字、贴纸、形状等）
      这里提供默认的装饰可以选择
       选择一个类型后，鼠标点击拖拽到照片墙上，就会在对应的位置添加装饰
        比如文字装饰：用户可以在照片墙的某个位置添加文字，文字可以是用户自己输入的，也可以是系统提供的文案
        比如贴纸装饰：用户可以在照片墙的某个位置添加贴纸，贴纸可以是用户自己上传的，也可以是系统提供的贴纸
        比如形状装饰：用户可以在照片墙的某个位置添加形状，形状可以是用户自己绘制的，也可以是系统提供的形状

  4. 导出选择

右侧就是预览情况
  上方是相册的页数，展示当前页和附近的页数 当前页数总是居中，用户点击切换到其他页时，会有一个平滑的过渡效果。
  下方是当前页的照片墙预览，展示用户添加的图片和装饰。
  照片墙的样式和布局，都可以由用户自己定义，用户可以在工具栏中选择不同的样式和布局，然后在预览中查看效果。

  照片墙放置之后，就可以点击照片墙（每个格子）上传图片。
  照片墙放置完毕，切换到装饰的时候就锁定，可以放置装饰的内容。

  悬浮一个画笔图案，点击后展示用户的历史操作记录，允许用户回到上一步或者下一步。
  右上角 按钮：操作导出（小字，大功告成） 按钮：分享（小字分享你的瞬间）。

# 扩写补充（保持原文风格，完善具体界面内容）

## A. 核心能力与价值主张
- 自由拼图：用户决定“哪些格子组成一个编号块”，布局更像自己。
- 一键模板：不想手动圈选的用户可以直接套用模板。
- 低门槛创作：按编号块填图即可完成作品。

## B. 详细用户流程（开始创作）
1) 进入编辑器 → 默认打开“相册选择”
2) 选择主题与尺寸 → 自动切换到“照片墙选择”
3) 选择模板布局或进入“我是建筑家”
4) 点击编号块上传图片 → 调整位置/缩放/旋转
5) 切换到“装饰” → 文字/贴纸/形状拖拽到画布
6) 保存/导出 → PNG 或整册 zip

## C. 左侧功能栏补充说明
### C1. 相册选择
- 主题卡展示：主题名称 + 简短描述 + 3-4 个代表色块。
- 尺寸选择：A4 / 9:16 / 1:1（后续可扩展自定义尺寸）。

### C2. 照片墙选择
- 提供 2 类入口：
  - **模板布局**：直接套用
  - **自定义布局**：进入“我是建筑家”
- 模板卡信息：布局缩略图 + 格子数量 + 推荐用途（毕业/团圆/旅行）。

### C3. 装饰选择
- 分类入口：文字 / 贴纸 / 形状 / 背景
- 交互规则：拖拽放置、点击选中后可调整大小与旋转
- 装饰层与图片层分离，避免误操作

### C4. 导出选择
- 清晰度：1x / 2x / 3x
- 提示：导出失败降级 1x，并提示用户
- 提供“导出进度条 + 预计耗时”提示

## D. 右侧预览区补充说明
- 顶部：页数导航（当前页居中，前后 2 页缩略）
- 中部：当前页画布（展示照片 + 装饰 + 编号块）
- 底部：操作提示条（例如：点击编号块上传图片）

## E. 自由拼图（我是建筑家）详细规则
1) 网格：将画布分成 N x M 个格子
2) 圈选：点击格子进行选择，选中格子高亮
3) 生成编号块：点击“生成区域”，合并为一个编号块
4) 编号递增：1, 2, 3... 自动生成
5) 冲突规则：已被占用的格子不可再选
6) 空格允许：可以保留空白格子
7) 保存布局：保存为“自定义模板”，可在模板库复用

## F. 模板库补充
- 默认模板至少覆盖：
  - 九宫格
  - 中心大图 + 周围小图
  - 爱心拼图
  - 时间轴拼图
- 用户自定义布局加入模板库

## G. 导出与分享
- PNG 单页导出
- zip 整册导出（每页命名）
- 分享按钮：复制链接 / 下载文件 / 生成分享卡（后续扩展）

## H. 验收标准（功能级）
- 用户能从模板库选择模板并上传照片完成拼图
- 用户能在“我是建筑家”中自由圈选生成编号块
- 编号块可单独上传与调整照片
- 可保存布局并再次使用
- 可成功导出 PNG 或 zip

## I. 开始创作页面组件清单（具体界面内容）
### I1. 顶部区域
- LOGO / 项目名称
- 主题切换器（默认主题 / 毕业 / 春节）
- 右侧主按钮：开始导出 / 分享（小字提示）

### I2. 左侧工具栏（固定宽度面板）
- 顶部 Tabs：相册选择 / 照片墙 / 装饰 / 导出
- 每个 Tab 内部：两列卡片布局（左列为分类，右列为详情）

#### Tab 1：相册选择
- 主题卡（大卡片）：名称 + 简介 + 代表色块 + 应用按钮
- 尺寸选项：A4 / 9:16 / 1:1
- 其他选项：背景纹理开关 / 是否显示页码

#### Tab 2：照片墙
- 模板库卡片：缩略图 + 格子数 + 推荐用途 + 一键应用
- 自定义入口：进入“我是建筑家”按钮
- 最近使用：最近 3 个模板快速入口

#### Tab 3：装饰
- 分类标签：文字 / 贴纸 / 形状 / 背景
- 文字样式：标题 / 副标题 / 小字说明
- 贴纸素材：主题贴纸 + 通用贴纸
- 形状素材：圆角矩形 / 圆形 / 线条 / 笔刷

#### Tab 4：导出
- 分辨率选择：1x / 2x / 3x
- 文件类型：PNG 单页 / ZIP 整册
- 导出按钮：主按钮 + 次按钮（禁用态、加载态）

### I3. 右侧画布区（预览 + 交互）
- 顶部页数导航：
  - 当前页居中，左右两侧可见邻页缩略
  - 支持点击切换，动效平滑
- 中部画布：
  - 网格背景 + 编号块显示
  - 点击编号块上传 / 替换照片
- 底部辅助条：
  - 操作提示（如：先圈选生成区域）
  - 快捷按钮（撤销 / 重做）

## J. “我是建筑家”页面交互状态清单
### J1. 初始状态
- 空画布提示：“请先选择尺寸并圈选格子”
- 默认网格预览（如 16x9）

### J2. 选中状态
- 选中格子高亮
- 顶部提示：“已圈选 X 个格子”

### J3. 生成区域状态
- 点击“生成区域”后生成编号块
- 已占用格子不可再选
- 生成后显示编号标签（1,2,3…）

### J4. 错误与阻止
- 非连通圈选：提示“区域不连通，请重新圈选”
- 重复圈选：提示“该格子已被占用”

### J5. 保存状态
- 保存成功提示：“已保存为自定义模板”
- 保存后出现在模板库“我的模板”区

### J6. 空白允许
- 未选格子保持空白，导出时为透明/纯色背景

### J7. 导出与应用
- “立即使用此布局”按钮
- 一键跳转编辑器并加载此布局

## K. 详细文案（按钮 / 提示 / 错误）
### K1. 按钮文案
- 主按钮：开始创作 / 生成区域 / 保存布局 / 导出 PNG / 导出整册
- 次按钮：清空圈选 / 删除区域 / 返回模板库 / 取消导出
- 工具按钮：撤销 / 重做 / 重置图片

### K2. 操作提示文案
- 引导提示：
  - “先圈选格子，再生成区域。”
  - “点击编号块上传照片。”
  - “拖动图片可调整位置，滚轮缩放。”
- 空状态提示：
  - “暂无布局，请先选择模板或创建布局。”
  - “暂无照片，点击编号块开始上传。”

### K3. 错误与阻止文案
- 圈选不连通：“区域不连通，请重新圈选。”
- 重复占用：“该格子已被占用。”
- 无选择：“请先圈选至少一个格子。”
- 导出失败：“导出失败，已自动降级为 1x，请重试。”

### K4. 成功与完成文案
- 保存成功：“已保存为自定义模板。”
- 导出完成：“导出完成，文件已下载。”
- 分享成功：“分享链接已复制。”

## L. 页面动效时序（进入 / 切换 / 导出）
### L1. 页面进入
- 页面进入时：背景渐入 + 卡片轻微上浮（motion-apple-slow）
- 画布区域延迟 120ms 进入，强调核心区域

### L2. Tab 切换
- Tab 切换：内容淡出 80ms → 内容淡入 160ms
- 选中 Tab：底部高亮条平滑滑入

### L3. 网格圈选与编号块
- 点击格子：高亮缩放 + 发光描边
- 生成区域：编号块弹性出现（轻微 scale-in）

### L4. 图片上传与调整
- 上传完成：图片淡入 + 轻微缩放
- 调整图片：拖拽实时更新位置

### L5. 导出流程
- 点击导出：按钮切换加载态 + 进度条出现
- 导出完成：进度条消失 + 成功提示淡入
- 导出失败：提示红色警示文案
