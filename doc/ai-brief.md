# AI Brief：马年新年主题·一键生成 H5 相册（小程序 + 豆包 AI）

> 用途：把这份 Brief 直接粘贴给不同“角色 AI”，要求其输出可执行交付物（设计/技术/运营），而不是泛泛建议。

## 1. 一句话目标
用户只需导入手机相册照片，豆包 AI 辅助生成图文与趣味符号，即可发布一个带动效与音乐的 H5 相册链接并分享。

## 2. MVP 主链路（必须极简）
选主题与模板 -> 批量导入照片 -> 自动按规则填充 -> AI 一键生成图文/符号（可重试、可编辑）-> 少量微调 -> 发布 -> 分享链接（微信内打开播放）。

## 3. 版本边界（先写死）
- 必做：小程序端创建与发布；发布产物为 H5 相册链接；H5 在微信内可打开播放；支持音乐与动效；生成封面海报图用于二次传播。
- 必做：AI 失败自动降级为模板默认文案，发布主链路不断。
- 后续：权限（仅自己/口令/过期）、数据统计（播放/转发/完播）、模板工坊、高级编辑。

## 4. 目标人群与价值主张（首版聚焦）
- 核心人群：愿意记录生活、重视审美与仪式感的用户（女性用户更高频）。
- 关键痛点：照片多但难整理、难成册、难“好看”、难快速分享。
- 核心承诺：不用会排版，导入就成册；不用会写文案，AI 一键出图文。

## 5. 主题优先级
首发三主题，但当前最重要主题为「马年新年（家人团聚）」；所有设计与 AI 规则优先围绕马年主题打磨到“当季爆款”质感。

## 6. 马年新年主题：视觉与资源包（首发必须）
### 6.1 视觉主调
- 氛围关键词：精致年味、喜庆但不俗、暖光团聚、轻奢金粉。
- 视觉主调：正红 + 香槟金 + 米白留白 + 暖黑压重（避免大面积纯红“糊”）。
- 贴纸元素池：灯笼、窗花、烟花、福字、祥云、春联角标、红包、马纹/马蹄印、奔跑线条、生肖印章。

### 6.2 配色建议
- Primary：#C81E1E（年味红）
- Accent：#F5C542（香槟金）
- Background：#FFF6E8（米白）
- Text：#1F1A17（暖黑）
- Highlight：#EAB308（亮金，仅小面积点亮）

### 6.3 纹理与装饰组件清单
- 纹理：金箔纹理、宣纸纹理、磨砂玻璃噪点、灯笼暖光渐变。
- 装饰组件：春联边框（上/下/左右）、福字角标、金粉描边框、窗花镂空框。
- 生肖模块：马年封面章（“马年/新春/生肖印”风格可选）、马元素边饰（不遮挡主体）。

### 6.4 封面样式（3 套）
- A「春联灯笼·团圆封面」
  - 构图：上下春联边框 + 中央主合影（圆角玻璃卡片）+ 右上福字角标
  - 文字：主标题 + 年份 + 地点（可选）
- B「金箔印章·轻奢封面」
  - 构图：米白大留白 + 中央大主图 + 金箔生肖印章角落点缀
  - 文字：短标题（8-12 字）+ 一句祝福（12-18 字）
- C「窗花拼贴·年味相册封面」
  - 构图：2-4 张小图拼贴叠层 + 窗花镂空框局部遮罩 + 一张主图压轴
  - 文字：标题 + 目录式副标题（如“年夜饭 / 拜年 / 合影”）

### 6.5 动效模板（3 套，轻量优先）
- T1「金粉入场」
  - 动作：主图淡入 + 轻推近；金粉低密度飘落；文字延迟出现
  - 适用：封面/留白文案/合影页
- T2「纸张翻页」
  - 动作：翻页 380-520ms + 纸张高光轻扫（极克制）
  - 适用：章节页/叙事页
- T3「烟花点亮」
  - 动作：角落烟花/光点点亮（不遮挡主体）+ 金色微光脉冲一次
  - 适用：拜年页/红包页/祝福页

### 6.6 音乐推荐风格（5 种）
- M1 国风轻鼓点：适合热闹合影/欢聚页
- M2 温暖弦乐与钢琴：适合团圆/留白文案页
- M3 新年 Lo-fi：适合拼贴/日常年味碎片页
- M4 传统器乐轻编：适合拜年/仪式感页
- M5 喜庆但克制：适合封面/结尾页

## 7. 照片“有机整合”（核心能力）
目标：把“很多张照片”变成“有结构、有节奏、有重点”的相册（8-12 页），避免用户陷入挑选与排版负担。

MVP（无需复杂识别也能显著提升成品感）：
- 自动分组：按时间聚类成 3-8 个章节（可手动合并/拆分）。
- 自动挑重点：每组挑 1 张主图 + 若干辅图（清晰度/曝光/时间间隔等规则）。
- 自动控密度：80-200 张压缩到 8-12 页，保证留白与呼吸感。
- 自动控重复：连拍/相似图去重（保留更清晰/更完整人像）。
- 自动编排节奏：主图页 -> 细节页 -> 合影页 -> 留白文案页循环。

## 8. AI（豆包）辅助创作：必须“像产品的一部分”
### 8.1 AI 输入
- 主题与模板
- 照片（默认上传压缩图用于理解；原图用于最终发布高清展示）
- 可选信息：年份、地点、家庭称呼、祝福倾向（可为空）

### 8.2 AI 输出（MVP 必做）
- 相册标题、副标题、封面一句话
- 每页段落文案（短文为主，避免大段长文）
- 图注候选（可一键替换）
- 趣味符号/标点装饰（与主题风格一致，克制）

### 8.3 主题 Prompt Pack（马年新年专属约束）
通用约束：
- 语气：温暖、精致、克制；不用土味口号、不过度鸡汤。
- 标点：优先中文标点；感叹号最多 1 个/页。
- 符号点缀：每页 0-2 个，避免刷屏。
- 意象池（按页分布，不要每页都重复）：灯笼、窗花、春联、烟花、金粉、祥云、马蹄/奔跑。
- 禁用：低俗/网感词、夸张营销词、PUA 语气、过度谐音梗。

PageType 输出规则：
- 封面：
  - 标题 8-14 字；副标题 12-20 字；祝福 12-24 字（可选）
  - 关键词：马年、新春、团圆、年味、喜乐、平安
- 合影：
  - 段落 20-45 字；图注 8-16 字（最多 2 条）
  - 关键词：团聚、相见、笑意、家常、这一刻
  - 要求：多写具体细节，少写空泛套话
- 年夜饭：
  - 段落 18-40 字；菜名/细节点 3-6 个短词（可选）
  - 关键词：烟火气、热气、香味、家宴、热腾腾
- 拜年：
  - 祝福主句 12-22 字；补一句 12-22 字（可选）；图注 8-14 字
  - 关键词：拜年、顺遂、平安、喜乐、万事如意
  - 避免：每页重复“新年快乐”
- 留白文案：
  - 文案 24-60 字（最多 2 段）；标题 6-10 字（可选）
  - 关键词：灯火、归家、温暖、慢慢来、好好生活

### 8.4 交互与失败降级
- 文风选择：温暖/幽默/克制/文艺（默认随主题）
- 强度滑杆：少量点缀 / 明显氛围
- 一键重试：保留结构/重写风格/更短/更活泼
- 输出校验：长度、敏感内容、重复度、主题一致性不达标则自动重试或降级为模板默认文案

## 9. 发布与 H5 播放器（必须好看且稳定）
- 发布结果：H5 相册链接（微信内可打开播放与转发）
- H5 播放器能力：动效播放、音乐开关、加载进度、封面与目录（可选）
- 首屏策略：封面 + 1-2 页优先加载，后续边播边预加载
- 降级策略：动效降级、音乐关闭、图片降清晰度
- 分享物料：自动生成封面海报（含标题与二维码/链接引导）

## 10. MVP 验收口径（给 AI 与团队对齐）
- 新用户从打开到发布并分享首个成品 ≤ 3 分钟
- H5 在微信内打开首屏加载目标 < 2s（常规网络口径）
- AI 生成失败不阻断流程：自动回退模板默认文案仍可发布
