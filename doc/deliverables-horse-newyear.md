# 交付物包：马年新年主题（小程序发布 H5 + 火山引擎 Doubao API（AK/SK 签名））

> 目标：把 PRD 变成可直接执行的“设计/技术/运营”交付物。默认口径：小程序端创建与发布；发布产物为 H5 相册链接；微信内可打开播放；动效与音乐可用；AI 失败不阻断发布。

## 0. 主管口径（写死的范围与验收）
### 0.1 MVP 范围（必须做）
- 主题：马年新年（优先），毕业/治愈可占位但不抢资源
- 主链路：选模板 -> 导入照片 -> 自动整合 -> AI 生成图文 -> 预览 -> 发布 -> 分享链接
- 发布产物：H5 链接 + 封面海报（含标题与二维码/链接引导）
- H5 播放：页切换动效、音乐开关、加载进度、首屏预加载与降级
- AI 兜底：生成失败自动回退模板默认文案，流程不断

### 0.2 MVP 验收（可量化）
- 首次使用：新用户从打开到发布并分享 ≤ 3 分钟
- 打开体验：H5 微信内首屏可见目标 < 2s（常规网络口径）
- 稳定性：AI/发布任一环节失败都能继续发布（降级策略生效）

### 0.3 默认决策（避免来回）
- 默认“轻奢年味”：喜庆但不俗；红/金小面积点亮；留白与层次优先
- 动效轻量优先：以 transform/opacity 为主，避免重粒子与大面积滤镜
- 音乐默认静音可播放：首次播放要求用户点击触发（符合浏览器限制）

### 1.4 动效模板（3 套）Motion Spec
统一约束：默认动效时长 450-650ms；封面 900-1200ms；仅 transform/opacity；低端机降级为“淡入”。

| 模板 | 关键动作 | 时长建议 | easing | 适用页面 | 降级 |
|---|---|---:|---|---|---|
| T1 金粉入场 | 主图淡入+轻推近；低密度金粉飘落；文字延迟出现 | 900-1200ms（封面）/ 500ms（页） | easeOutCubic | 封面/留白/合影 | 关闭粒子，只保留淡入 |
| T2 纸张翻页 | 翻页过渡 + 轻高光扫过 | 380-520ms | easeInOutCubic | 章节/叙事 | 改为左右滑动淡入 |
| T3 烟花点亮 | 角落光点点亮 + 一次微光脉冲 | 600-900ms | easeOutQuad | 拜年/红包/祝福 | 关闭光点，只保留亮度渐变 |

## 2) 技术交付物（可直接给研发/架构）

### 2.1 端到端数据流（文字分层）
**小程序端**
1) 选择模板（templateId）与导入照片（localPath[]）
2) 生成 Asset：创建缩略图（thumb）、提取基础元数据（尺寸/大小/可选 EXIF）
3) 照片整合：时间聚类 -> 章节 -> 选主图/去重 -> 编排页序
4) AI 生成：把“模板结构 + pageType + 选图摘要 + 可选用户信息”发给后端 AI 服务
5) 预览确认：本地预览或 WebView 预览（可选）
6) 发布：上传多规格图片 + 生成 album.json -> 获取 shareUrl + posterUrl

**后端/云函数**
1) 接收上传（多规格图片、音乐引用、album 草稿）
2) 调用火山引擎 Doubao API（AK/SK 签名）（或复用缓存），产出文案字段
3) 内容安全（AI 文案审核 + 图片审核策略）
4) 生成并固化 album.json（签名/版本/资源 URL）
5) 返回 shareUrl、posterUrl

**H5 播放器**
1) 加载 album.json（首屏封面 + 1-2 页资源优先）
2) 渲染 pageType 对应模板（动效/音乐/预加载/降级）
3) 埋点上报（打开/完播/分享点击）

### 2.2 对象存储目录结构（建议）
以 albumId 分区，避免单目录膨胀：
```
/albums/{albumId}/
  album.json
  poster.jpg
  images/
    original/{assetId}.jpg
    large/{assetId}.jpg
    medium/{assetId}.jpg
    thumb/{assetId}.jpg
  audio/
    {trackId}.mp3
  previews/
    cover.jpg
    page-001.jpg
```

### 2.3 album.json（Schema：最小可用口径）
说明：这是“播放器协议”。小程序发布时生成，H5 只按此渲染。

**类型表**
| 字段 | 类型 | 必填 | 说明 |
|---|---|---:|---|
| version | number | 是 | 协议版本 |
| albumId | string | 是 | 全局 ID |
| themeId | string | 是 | horseNewYear |
| title | string | 是 | 相册标题 |
| music | object | 否 | 背景音乐 |
| motionPreset | string | 否 | T1/T2/T3 |
| pages | array | 是 | 页面列表 |

**Page**
| 字段 | 类型 | 必填 | 说明 |
|---|---|---:|---|
| pageId | string | 是 | |
| pageType | string | 是 | cover/groupPhoto/dinner/greeting/blank |
| layoutId | string | 是 | 布局模板 |
| assets | array | 是 | 本页用到的图片资源 |
| text | object | 否 | AI 文案结果（可为空/降级） |
| decorations | array | 否 | 贴纸/框/印章等 |

### 2.5 API/云函数清单（MVP）
| 名称 | 作用 | 入参（简） | 出参（简） | 失败降级 |
|---|---|---|---|---|
| CreateDraftAlbum | 创建草稿 | templateId, assetMeta[] | albumId | 本地草稿 |
| GenerateAIText | 调用火山引擎 Doubao API（AK/SK 签名）生成文案 | albumId, pageSummaries, style | pageText[] | 回退模板默认文案 |
| UploadAssets | 上传多规格图片 | albumId, files[] | urls[] | 仅上传 thumb/medium |
| PublishAlbum | 固化并发布 | albumId, selectedMusic, motionPreset | shareUrl, posterUrl | 发布为静态无音乐/无动效 |
| GetAlbum | H5 拉取 | albumId | album.json | 返回缓存版本 |

### 2.6 内容安全与隐私（MVP）
- 文案审核：AI 输出先过审核；不通过则替换为模板默认文案（不中断发布）
- 图片审核：首版可“抽样 + 举报”策略；后续接入全量审核或敏感场景开关
- 隐私口径：用于 AI 理解的图片默认使用压缩图；提供关闭 AI 的纯模板模式
- 日志最小化：不记录原图；仅记录 assetId、尺寸、哈希、审核结果与耗时

## 4) 风险清单（默认应对）
| 风险 | 影响 | 默认应对 |
|---|---|---|
| H5 首屏慢 | 分享转化下降 | 首屏只加载封面+1页；图片多规格；CDN |
| 音乐无法自动播 | 体验落差 | 默认静音；明显提示“点一下开启音乐” |
| AI 输出不稳定 | 内容翻车 | Prompt Pack + 输出校验 + 自动重试 + 默认文案回退 |
| 主题跑偏/土味 | 影响审美定位 | 禁用词/长度约束/符号上限；克制风格强约束 |
| 发布失败 | 主链路断 | 断点续传 + 重试 + 降级发布（无音乐/无动效） |
