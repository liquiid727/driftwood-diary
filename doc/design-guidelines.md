# 基础视觉与动效规则

本规则定义为全站默认基调（动森式配色氛围 + 苹果式过渡），用于所有页面与组件，不作为首页独立模块展示。

## 1. 基础配色（动森式氛围）
- 色彩来源：`src/app/globals.css` 的 CSS 变量（`--paper`, `--cream`, `--rose`, `--sky`, `--ink`, `--muted`）。
- 氛围层：使用 `bg-hero` 作为页面背景基调，配合 `glass` / `glass-strong` 形成柔和玻璃质感。
- 允许渐变：可使用 `--hero-*` 变量构建柔和渐变与光晕背景。

## 2. 过渡与动效（苹果式平滑）
- 默认过渡：以 `opacity` / `transform` 为主，避免复杂位移。
- 推荐节奏：`--motion-fast` / `--motion-base` / `--motion-slow` 作为全局时长规范。
- 推荐缓动：`--ease-apple` 作为统一缓动曲线。
- 可访问性：尊重 `prefers-reduced-motion`，必要时降级动效。

## 3. 首页展示原则
- 首页不再使用「动森式配色氛围」「苹果式过渡」作为独立模块。
- 该风格仅作为全站默认设计准则贯穿组件与页面。
